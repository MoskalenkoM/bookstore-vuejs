Необходимо создать адаптивное web приложение книжного магазина на REACT или VUEJS. На первой странице выводится список с книгами (pic1.png).
При клике по элементу в списке - переход на страницу с редактированием или удалением, где все поля заблокированы для измененения (pic2.png).
При клике на удаление книга удаляется (можно вывести предупреждение перед удалением). При клике на редактирование - поля становятся доступны для изменений (pic3.png).
Можно изменить и сохранить изменения. Страница с добавлением новой книги, на ней все поля доступны сразу для изменений (pic4.png).
